1. Por que o arquivo .env não deve ser versionado?

O arquivo .env não deve ser versionado porque contém informações sensíveis, como chaves secretas e configurações privadas, que podem comprometer a segurança do sistema.

2. Qual a função do pip freeze?

O comando pip freeze serve para listar todas as dependências do projeto com suas versões, permitindo recriar o ambiente em outro computador.

3. O que pode acontecer se a SECRET_KEY for exposta?

Se a SECRET_KEY for exposta, um atacante pode comprometer a segurança da aplicação, falsificar sessões e gerar acessos indevidos.

4. Qual o papel do .gitignore?

O .gitignore serve para impedir que arquivos desnecessários ou sensíveis, como o .env e o venv, sejam enviados para o repositório.
